Jezici Frontend razvoja

Za razvoj Frontenda, koriste se osnovna tri jezika:

  1. HTML
  2. CSS
  3. JavaScript

U nastavku se nalazi kratak opis svakog jezika.

HTML

HTML skraceno od HyperText Markup Language, je jezik koji se koristi za kreiranje strukture dokumenta i skupa informacija jednog web sajta/aplikacije.

CSS

CSS skraceno od Cascading Style Sheets, je jezik koji se koristi za stilizaciju i izgled HTML dokumenta.

JavaScript

JavaScript je programski jezik koji se koristi za funkcionalnost web sajta/aplikacije. Spada u jezike viseg nivoa i kljucan je za Frontend razvoj.
